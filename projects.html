Smart India Hackathon 2024

Proposed solution ‚Äî The objective focuses on implementing an algorithm for Automatic Modulation Recognition of DVB-S2X waveforms.

A custom GNU Radio algorithm incorporates a machine learning model, enabling real-time analysis of DVB-S2X signals through an Embedded Python Block.

Employing Software Defined Radio for efficient signal reception and real-time data analysis to minimize hardware requirements.

The developed machine learning model achieves 91% accuracy in DVB-S2X modulation recognition across a wide range of SNR, leveraging CNNs for feature extraction and LSTMs for sequence prediction.

The solution enables scalable detection of DVB-S2X waveforms by integrating diverse modulation schemes into a streamlined algorithm, ensuring adaptability to future advancements sustainably.

Model specifications:

  • Data Loading & Preprocessing: CSV files are loaded and reshaped into 64x32 matrices for I and Q components. The data is split into training and test sets with categorical labels.
  • CNN Layers: Two convolutional layers per input (64 filters), followed by max-pooling, then flattened.
  • Concatenation of I and Q features.
  • LSTM Layer: 256-unit LSTM processes the features.
  • Dense Layers: 1024, 256, 32 units with LeakyReLU and 50% dropout.
  • Output: Softmax layer for classification.
  • Training: Adam optimizer, 25 epochs, batch size 64.
  • Evaluation: Accuracy vs SNR plot for visualization.
